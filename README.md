### Introduction

A comprehensive combat script for Blox Fruits, designed for high performance and maximum security. The script utilizes an advanced loader architecture to ensure stability and protect its features.

| Category | Feature | Description |
| :--- | :--- | :--- |
| **Combat** | **Silent Aim** | Aims without moving your camera. (Raycast-based) |
| | **Camera Aimbot** | Smoothly moves your camera to the target when holding Right Mouse Button. |
| | **Advanced Targeting** | Prioritizes targets based on Level, Type (Player/Monster), Health, and Distance. |
| | **Wall Check** | Option to ignore targets behind walls. |
| **Visuals** | **Full ESP** | ESP for Players and Monsters with customizable information. |
| | **Tracers** | Draws lines to targets from various origin points (Center, Mouse, Bottom). |

## 🚀 How to Use

This script uses a secure loader system. You only need to execute the loader script.

1.  Copy the entire content of the **loader script**.
2.  Open your executor in-game.
3.  Paste and execute the script.

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/HashimotoAkira/HashimotoAkira-BloxFruitPvpHub/refs/heads/main/BloxFruitPvpLoader"))()
```

> **Note:** For best stability, please wait 1-2 minutes after joining the game and selecting your team before executing the script. This allows the game to fully load all assets and helps prevent potential UI conflicts or errors.
